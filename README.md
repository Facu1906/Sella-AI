# Sella — WhatsApp AI Sales Agent

AI agent that converts WhatsApp conversations into purchases automatically. Built for e-commerce brands.

**Live demo:** [whatsapp-sales.vercel.app](https://whatsapp-sales.vercel.app)

## What it does

- Receives WhatsApp messages from customers
- Asks 1-2 qualifying questions
- Recommends specific products from the catalog
- Handles objections (price, sizing, shipping)
- Pushes customer to checkout

## Stack

- Claude API (Anthropic)
- Node.js / Express
- WhatsApp Business API
